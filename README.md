# 🧮 C Calculator CLI Project

A simple console-based calculator in C that performs:

- Addition
- Subtraction
- Multiplication
- Division

### 👩‍💻 How to Use:
1. Compile using `gcc calculator.c -o calc`
2. Run using `./calc`
3. Follow on-screen instructions!

✨ Created by Maria Ali as a beginner C project.
